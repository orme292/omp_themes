# oh-my-posh themes

### orme10k

Based on the p10k theme for oh-my-zsh

Supports:

- git
- python (venv)
- pyproject.toml
- ssh session identifiers
- root session identifiers

orme10k-short is the shorter version, for the smaller screens

- git
- python (venv)
- ssh session identifiers
- root session identifiers
